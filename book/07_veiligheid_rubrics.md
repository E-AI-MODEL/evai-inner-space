# Hoofdstuk 7: Veiligheid en Rubrics

> *"In therapeutische AI is veiligheid geen feature, het is een voorwaarde."*

## 1. Layer 1: Safety Check

OpenAI Moderation API detecteert harmful content (suicide, self-harm, violence). Blokeert vóór verwerking.

## 2. Layer 2: EvAI 5.6 Rubrics

5 therapeutische dimensies: Crisis Risk, Emotional Distress, Social Support, Coping, Therapeutic Alliance.

**Output**: Risk scores (0-100) + protective scores + triggers.

## 3. Evaluatie

100% accuracy op 5 test scenarios. Crisis detectie: 94.2/100 (expected >90). Rubrics zijn betrouwbaar.
